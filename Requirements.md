Requirements
=====

This Crawler requires the following libraries installed in order to compile it.

| Name | Install (Tested on Ubuntu 14.10) |
| ---- | ---- |
| MySQL Connector | `apt-get install libmysqlcppconn-dev` |
| Lib cURL | `apt-get install libcurl4-openssl-dev` |
| Lib cURLpp | `apt-get install libcurlpp-dev` |
| Lib JSON | `apt-get install libjsoncpp-dev` |
